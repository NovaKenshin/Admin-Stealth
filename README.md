# 🛡️ Admin-Stealth

**Admin-Stealth**, sunucu yöneticilerinin spectator (izleyici) modundayken oyuncular tarafından fark edilmeden oyun takibi yapmalarını sağlayan, sunucu yönetimine profesyonel bir "görünmezlik" katan eklentidir.

## ✨ Özellikler
* **Tam Gizlilik:** Aktif edildiğinde, adminler spectator listesinde tamamen gizlenir.
* **Kolay Yönetim:** Basit komutlarla anlık olarak gizlilik modunu değiştirebilirsiniz.
* **Hafif ve Hızlı:** Sunucu performansına hiçbir etkisi yoktur.

## 🚀 Komutlar

Admin-Stealth'in kullanımı oldukça pratiktir. Yetkili olduğunuz sürece aşağıdaki komutları oyun içi sohbetten kullanabilirsiniz veya Console Kısmından

Spectator Kısmına geçip Komutu Kullanabilirsiniz.
| Komut | Açıklama |
| :--- | :--- |
| `css_gizle` | Admini spectator listesinden gizler. |
| `css_ungizle` | Admini tekrar görünür hale getirir (gizlemeyi kaldırır). |

# 🛡️ Admin-Stealth

**Admin-Stealth** is a professional plugin designed for server administrators to monitor gameplay while remaining completely invisible to other players, even when in spectator mode.

## ✨ Features
* **Full Privacy:** Once activated, admins are completely hidden from the spectator list.
* **Easy Management:** Toggle stealth mode instantly with simple commands.
* **Lightweight & Fast:** Optimized for performance; it has no impact on server stability.

## 🚀 Commands

Admin-Stealth is very practical to use. As long as you have the required permissions, you can use the following commands via in-game chat or the console.

*Note: You can use these commands while in spectator mode.*

| Command | Description |
| :--- | :--- |
| `css_gizle` | Hides the admin from the spectator list. |
| `css_ungizle` | Makes the admin visible again (disables stealth mode). |

## ⚙️ Installation
1. Upload the plugin files to the designated directory on your server.
2. Restart your server or reload the plugin.
3. Ensure that authorized users have the required permissions configured.

**Required Permission (Flag):** `@css/ban`
